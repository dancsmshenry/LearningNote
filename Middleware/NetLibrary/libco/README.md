# 前置知识

## 寄存器

- 以常见的64位处理器作为例子

- | 标号 | 作用         |
  | ---- | ------------ |
  | %rax | 返回值       |
  | %rbx | 被调用者保存 |
  | %rcx | 第4个参数    |
  | %rdx | 第3个参数    |
  | %rsi | 第2个参数    |
  | %rdi | 第1个参数    |
  | %rbp | 被调用者保存 |
  | %rsp | 栈指针       |
  | %r8  | 第5个参数    |
  | %r9  | 第6个参数    |
  | %r10 | 调用者保存   |
  | %r11 | 调用者保存   |
  | %r12 | 被调用者保存 |
  | %r13 | 被调用者保存 |
  | %r14 | 被调用者保存 |
  | %r15 | 被调用者保存 |

- 而协程的切换，不需要考虑%r10和%r11这两个寄存器上的值
