# relational languages

- Data mainpulation language 数据的增删改查，数据操纵语言（DML）
- Data definition language 数据的定义语言，数据库的create，元数据操作（DDL）
- Data control language 数据的控制语言，像权限的管理（DCL）



- 还可以用sql定义视图，定义权限，开始事务
